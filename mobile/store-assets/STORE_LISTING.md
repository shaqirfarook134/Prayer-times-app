Stay connected with your local masjid and maintain your prayer schedule with confidence. Our app is specifically designed for the Al Taqwa Masjid community, ensuring you always have access to the most accurate prayer times.

## Notes for Hosting Privacy Policy

You need to host the privacy policy at a public URL. Options:

1. **GitHub Pages** (Free)
   - Create a public GitHub repo
   - Enable GitHub Pages in settings
   - Upload PRIVACY_POLICY.md
   - URL: https://[username].github.io/[repo]/PRIVACY_POLICY.html

2. **Google Sites** (Free)
   - Go to sites.google.com
   - Create a new site
   - Copy/paste privacy policy content
   - Publish and copy URL

3. **Masjid Website**
   - If Al Taqwa Masjid has a website, host it there
   - Example: https://altaqwamasjid.org/app-privacy-policy

4. **Netlify/Vercel** (Free)
   - Deploy a simple static site
   - Upload privacy policy as HTML

Choose the easiest option for you. I recommend GitHub Pages if you already have a GitHub account.
